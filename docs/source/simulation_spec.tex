\section{Introduction}

The simulation spec file contains the entire configuration of a simulation. It
is in \href{json}{https://en.wikipedia.org/wiki/Json} format and divided into
two main sections: ``assignment,'' and ``configuration.''

\section{Assignment}

The assignment section is primarily relevant for use with EGTA. It contains a
